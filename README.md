# Super Gameboy Quest - Toadzillart | Yuki | kuyaed

A retro-inspired 2D dungeon crawler built with **PixiJS** and **TypeScript**.
This project serves as a technical demonstration of modern web game development,
automated deployment, and high-performance rendering.

## Live Demo

[Play Game Here](https://kuyaed-1720.github.io/my-pixi-game/)

---

## Technical Stack

- **Engine:** PixiJS (WebGL-based 2D rendering)
- **Language:** TypeScript
- **Build Tools:** Vite (Fast bundling and HMR)
- **Automations:** GitHub Actions (CI/CD Pipeline for automated deployment)

## Key Features (Current)

- **Tilemap Rendering:** Efficiently rendering dungeon environments using PixiJS Containers.
- **Player Controller:** Fluid 8-way movement and animation state handling.
- **Automated Pipeline:** Every push to `main` automatically builds and deploys the latest version to GitHub Pages.

## Local Setup

1. Clone the repo: `git clone https://github.com/kuyaed-1720/my-pixi-game.git`
2. Install dependencies: `npm install`
3. Run dev server `npm run dev`
4. Build for production: `npm run build`
